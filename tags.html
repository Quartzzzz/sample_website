Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell
header header represent a container for introductory content or a set of navigational links
div division define a division or a section in an HTML document
ol ordered list create an ordered list
ul unordered list create an unordered list
li list create a line within a list

Inline Elements

Tag Name Purpose Example Result
em emphasize emphasize a string of text <em>a string of text</em> a string of text
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code include a code <code>some code</code> some code
span span group inline-elements in a document <span>some inline-elements</span> void

Document tags

Tag Name Purpose
htmlhtml html indicate to browser that it is an HTML document, contain all other HTML tags except <!DOCTYPE>
head head contain title and metadata relative to the page
body body contain all the contents of a HTML page
title title contain title displayed on Tab in browser, favorites and search-engine result
meta meta contain link to other files (script,css), information about author, last modification

Styling Elements

CSS element Purpose Value example
text-align Set the text alignment for different <div> elements text-align: center, right, left
text-decoration specify the decoration added to text text-decoration: none
margin create space around elements, outside of any defined borders. margin: (top)px (right)px (bottom)px (left)px
padding generate space around an element's content, inside of any defined borders padding: (top)px (right)px (bottom)px (left)px
float specifies how an element should float. Elements after a floating element will flow around it float: left, right
display specify the display behavior (the type of rendering box) of an element display : block, inline
